# Turbo Merch - AI-Powered Merchandise Platform

This is an AI-powered merchandise platform that helps you discover trends, generate product listings, and manage your inventory using Google's Gemini AI.

## Features

- **Trend Scanner**: Discover trending topics and keywords
- **Listing Generator**: Create optimized product listings with AI
- **Design Research**: Gather design inspiration and research
- **Library**: Save and manage your generated listings

View your app in AI Studio: https://ai.studio/apps/drive/1Iosu555lL_dJb5rtKJoK95ImVf0Luz0L

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
